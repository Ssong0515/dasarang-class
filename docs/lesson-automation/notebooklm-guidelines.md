# NotebookLM 슬라이드 공통 지침

이론 슬라이드는 NotebookLM으로 제작한다(NotebookLM은 공개 API가 없어 자동화 대상이 아님 — 수동).
아래 **두 지침을 NotebookLM 출력 커스터마이즈(지침) 칸에 한 번 넣어두고**, 매번 시수별 **내용 프롬프트**만 추가로 넣는다.

대상 학습자: 다문화 이주민 성인 디지털 문해 초급(컴퓨터를 처음 다루고, 한국어를 배우는 중이며, 어르신도 포함).

---

## ① 디자인 지침

```
[슬라이드 디자인 지침 — 공통]
대상은 한국어가 서툰 외국 출신 성인·어르신 학습자입니다. 한눈에 이해되는 슬라이드를 만드세요.

1. 한 슬라이드 = 한 메시지. 욕심내지 말고 비웁니다.
2. 글자는 최소화. 슬라이드당 핵심 문장 1~2개(각 10~15자). 긴 설명·문단 금지.
3. 아주 큰 글씨. 제목은 화면을 꽉 채울 만큼, 본문도 멀리서 읽힐 크기. 작은 글씨 금지.
4. 글보다 그림. 모든 슬라이드에 내용을 설명하는 큰 그림·아이콘·실물 사진을 1개 이상. 추상적 클립아트보다 또렷한 실물·아이콘.
5. 색: 흰색(또는 아주 연한색) 배경 + 진한 글자(검정·진남색). 강조는 슬라이드당 한 색만. 형광색·여러 색 혼용 금지. 색맹·저시력 고려해 대비를 높게.
6. 글꼴: 굵고 단순한 고딕체(맑은 고딕 류). 장식체·흘림체 금지. '굵게'로 핵심만 강조.
7. 레이아웃 통일: 제목은 항상 위, 그림은 가운데, 짧은 설명은 아래. 모든 슬라이드 같은 틀.
8. 순서·동작은 번호(①②③)와 화살표(→)로 시각화. 글로 나열하지 않습니다.
9. 여백을 넉넉히. 모서리까지 꽉 채우지 않습니다.
10. 진행 표시(예: 3/10)로 현재 위치를 알려줍니다.
11. 화면 전환·애니메이션은 최소. 빠르거나 어지러운 효과 금지.
12. 이모지·아이콘은 의미가 분명한 것만, 위치를 일관되게.
```

## ② 내용·구성 지침 (언어·말투·흐름·교육 원칙)

```
[슬라이드 내용·구성 지침 — 공통]
대상: 다문화 이주민 성인 디지털 문해 초급(컴퓨터를 처음 다루고, 한국어를 배우는 중이며, 어르신도 포함).

A. 언어
- 아주 쉬운 한국어만. 한 문장은 짧게(10~15자), 한 슬라이드 2문장 이내.
- 초급 어휘 사용. 어려운 말·외래어·전문용어는 쉬운 말로 풀고, 꼭 필요하면 괄호로 쉬운 설명을 붙입니다.
- 부드러운 말투("~해요", "~봐요"), 높임말. 학습자는 "여러분"으로 부릅니다.
- 핵심 단어는 학습자 모국어 병기를 넣을 수 있게 별도 목록으로도 정리(선택).

B. 말투·태도
- 따뜻하고 격려하는 말투. "틀려도 괜찮아요", "참 잘했어요".
- 천천히, 한 번에 하나씩. 재촉하지 않습니다.
- 학습자의 출신·문화를 존중. 종교·정치 등 민감 주제, 특정 국가 비하·고정관념 금지.

C. 구성(흐름)
- 도입(1~2장): 오늘 배울 것 + 왜 필요한지(실생활 이유).
- 본문: 개념을 하나씩. 각 개념은 (무엇인지 → 언제 쓰는지 → 어떻게 하는지) 순서.
- 실생활 예: 한국 생활·센터·가족·휴대폰 등 학습자 맥락의 예시를 듭니다.
- 중간 확인: 아주 쉬운 질문이나 따라하기 1개.
- 정리(1장): 오늘 배운 것을 3가지 이내로 요약.
- 마무리(1장): "이제 직접 해봐요"로, 이어지는 인터랙티브 실습으로 연결.

D. 교육 원칙
- 한 슬라이드 한 개념. 이전에 배운 것을 짧게 복습·연결.
- 비유와 실물로 설명(예: 폴더 = 서랍).
- 분량: 한 시수당 8~12장.

E. 정확·안전
- 사실만 전달하고, 모르면 지어내지 않습니다.
- 개인정보·비밀번호·결제 등은 항상 "조심하기"를 강조합니다.
```

## 시수별 내용 프롬프트 (매번 따로 입력 — 예시 템플릿)

지침 ①②는 고정. 아래만 시수마다 갈아끼운다. 값은 해당 회차 `details`에서 가져온다.

```
이번 시수의 이론 슬라이드를 만들어 주세요.
- 회차/시수: [예: 1회차 2시수]
- 주제: [예: 컴퓨터와 마우스 미션]
- 모듈: [예: 시작하기]
- 주요 활동: [예: 전원, 마우스, 클릭, 더블클릭, 드래그]
- 이 시수의 결과물: [예: 기본 조작 미션표]
이어지는 실습(인터랙티브 게임)으로 연결되게 마무리해 주세요.
```
